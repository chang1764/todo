# 04 — 차별화·성취감

## 선택된 기능

### E1: 팀 대시보드

**목표**: 팀의 활동을 한눈에 보면서 협력 분위기 조성

**구체적 구현**
- 보드 상단에 "이번 주 팀 통계" 위젯
  - 팀원별 완료 카드 수 (막대 또는 리스트)
  - 가장 활발한 팀원 (이모지 뱃지)
  - 팀 전체 완료율 (%)
  - 기한 초과 카드 수 경고
- 클릭하면 상세 대시보드 모달 (팀원별 활동 흐름, 요일별 완료 그래프)

**효과**
- 리더(박준호): 팀의 진행 상황을 수치로 빠르게 파악 → 스탠드업 진행 용이
- 팀원(김다현): "나도 저 사람처럼 활발하게" 동기 부여 → 지속적 참여
- 팀 전체: 협력 문화 강화 → "우리 함께하는 느낌"

---

### E2: 카드 완료 애니메이션

**목표**: 카드를 Done으로 옮길 때 즉각적인 만족감 제공

**구체적 구현**
- 카드를 Done 컬럼으로 드래그했을 때
  - 카드가 초록색으로 변하면서 체크 아이콘 출현
  - 불꽃/파티클 애니메이션 (1초)
  - "완료!" 텍스트 팝업
- 모바일에서도 같은 효과 (끊김 없이)
- 애니메이션 후 카드가 Done 컬럼에 안착

**효과**
- 팀원(김다현): 매번 작업 완료 시 작은 쾌감 → "이 앱 쓸 맛이 난다"
- 반복 사용 증대: 매일 최소 1개 이상 Done 카드 만들려는 심리
- 팀 분위기: 다른 팀원이 완료하는 것을 보면서 자극받음

---

### E3: 기한 임박 알림 (12시간 전)

**목표**: 기한 놓치는 것 방지 + 긴장감 유지

**구체적 구현**
- 카드 기한이 12시간 남았을 때
  - 해당 카드가 빨간색/주황색으로 변함
  - 보드 우측 상단에 "⏰ 기한 임박: 3개" 배너
  - 클릭하면 기한 임박 카드 목록 표시
- 웹 알림 (브라우저 알림 on/off 선택 가능)
- 카드 우선도가 자동으로 높아지지는 않음 (사용자가 수동으로 조정)

**효과**
- 리더(박준호): 기한 관리가 수동이 아닌 반자동 → 막판 대처 시간 확보
- 팀원(김다현): "어, 내일까지네?" 다시 인지 → 마무리 속도 올림
- 팀 전체: 기한 초과 사건 줄임

## 의도적으로 넣지 않는 것

- ❌ **팀원 랭킹** — "1위가 뭐하는 사람이야?" 불편한 비교 유발 (협력 방해)
- ❌ **카드 댓글 알림** — 반나절 예산 초과, 추후 v2에서 추가
- ❌ **자동 우선도 조정** — 사용자 주권 침해 (리더가 수동으로 판단)
- ❌ **일일 목표** — 팀마다 다르고 강제력이 너무 강함
- ❌ **반복 카드** — 반나절 예산 내에서 DB 설계 복잡화
- ❌ **팀 채팅** — Slack 등과 중복, 칸반 본질과 분산

## 데이터 모델 추가

**statistics 테이블** (대시보드 통계용)
```
id: UUID
week_start: Date (주 시작일)
user_id: String (팀원 이름, 인증 없으므로 간단한 식별자)
completed_count: Integer (이번 주 완료 카드)
updated_at: Timestamp
```

**notifications 테이블** (기한 알림 추적용)
```
id: UUID
card_id: UUID
notified_at: Timestamp (언제 알림 보냈는지)
dismissed: Boolean (사용자가 읽었는지)
```

**M1-M2** (필수)

**M3-M4** (선택)
