# 05 — 디자인

## 디자인 개요

**시안**: 시안 2 (사이드바 + 중앙 칸반 + 대시보드) — 모바일 최적화
**분위기**: 명랑 (친근, 컬러풀, 팀 분위기)
**참고**: Jira (전문적 레이아웃, 통계 강화)

---

## 레이아웃 구조

### 데스크톱 (1200px 이상)

```
┌─────────────────────────────────────────────────────┐
│ 헤더 (로고, 검색, 사용자)                              │
├────────┬─────────────────────────┬──────────────────┤
│        │                         │                  │
│ 사이드 │   칸반 보드             │  팀 대시보드      │
│ 바     │   (To Do / In Prog /   │  (통계, 활발팀원 │
│        │    Review / Done)      │   기한 임박)      │
│        │                         │                  │
│        │   - 컬럼 4개~5개        │                  │
│        │   - 카드 드래그         │  - 이번주 완료   │
│        │   - 우선도 색상         │  - 진행중 개수   │
│        │   - 기한 표시           │  - 가장 활발팀원 │
│        │                         │  - 기한 임박     │
└────────┴─────────────────────────┴──────────────────┘
```

**사이드바 (왼쪽, 200px 고정)**
- 내 칸반
- 즐겨찾기
- 필터
- 통계
- 설정

**칸반 보드 (중앙, 가변)**
- 컬럼: To Do, In Progress, Review, Done (+ 커스텀 추가 가능)
- 카드: 제목 + 우선도 색상 + 기한 아이콘
- 드래그 & 드롭으로 이동

**대시보드 (오른쪽, 250px 고정)**
- 이번 주 완료 카드 수
- 진행 중 카드 수
- 가장 활발한 팀원 (이모지)
- 기한 임박 카드 경고

### 태블릿 (768px ~ 1200px)

```
┌─────────────────────────────────┐
│ 헤더 (햄버거 메뉴)               │
├─────────────────────────────────┤
│   칸반 보드                      │
│   (2개 컬럼 visible + 스크롤)   │
│                                 │
│   (사이드바: collapse 가능)      │
│   (대시보드: 아래로 스크롤)      │
└─────────────────────────────────┘
```

- 사이드바 collapse (햄버거 메뉴 토글)
- 칸반 보드: 2개 컬럼만 보이도록 (좌우 스크롤)
- 대시보드: 하단에 표시

### 모바일 (<768px)

```
┌─────────────────────────────────┐
│ 헤더 (메뉴 버튼, 보드명)         │
├─────────────────────────────────┤
│                                 │
│   칸반 보드                      │
│   (1개 컬럼 visible + 좌우 스크롤│
│                                 │
│                                 │
├─────────────────────────────────┤
│ 탭: 보드 | 대시보드 | 설정      │
└─────────────────────────────────┘
```

- 사이드바 숨김 (필요시 슬라이드 메뉴)
- 칸반 보드: 1개 컬럼만 보이도록 (좌우 스크롤)
- 하단 탭: 보드 / 대시보드 / 설정 전환
- 대시보드: 탭으로 분리 표시

## 컬러 팔레트

### 우선도 색상
- **High (🔴)**: #FF6B6B (빨강)
- **Medium (🟡)**: #FFE66D (노랑)
- **Low (🟢)**: #95E1D3 (초록)

### 상태 색상
- **To Do**: #FF6B6B (빨강 상단)
- **In Progress**: #4ECDC4 (청록 상단)
- **Review**: #FFE66D (노랑 상단)
- **Done**: #95E1D3 (초록 상단)

### 배경 & 중립
- **주색상**: #667EEA (보라, 링크/버튼)
- **배경**: #F5F7FA (밝은 회색)
- **카드 배경**: #FFFFFF (흰색)
- **텍스트**: #333333 (진한 회색)
- **보조 텍스트**: #999999 (중간 회색)

### 명랑 분위기
- 밝고 따뜻한 톤 (파스텔 X, 생생함 O)
- 카드: 흰색 배경 + 상단 컬러 테두리
- 여백: 충분함 (답답하지 않게)

- 작은 텍스트 (기한, 태그): 12px, 400

## 컴포넌트 명세

### 1. 칸반 카드

```
┌─────────────────────────┐
│ 제목 (13px, 500)        │
│ 설명 (11px, 400) - opt  │
├─────────────────────────┤
│ 🔴 High • 내일 • [👤]   │
│ 기한 임박 (12시간 전)   │
└─────────────────────────┘
```

- 좌측 3px 보더: 우선도 색상
- 기한 12시간 전: 우측 상단 ⏰ 아이콘 + 빨강 배경
- 완료 애니메이션: Done 이동 시 1초 불꽃 + 체크마크
- 호버: 그림자 강화 (0 4px 12px rgba(0,0,0,0.15))

### 2. 칸반 컬럼

```
┌────────────────────┐
│ 📋 To Do      [3]  │
├────────────────────┤
│ [카드]             │
│ [카드]             │
│ [+ 새 카드]        │
└────────────────────┘
```

- 헤더: 컬럼명 + 카드 수 (배지)
- 상단 4px 보더: 상태 색상
- 최소 너비: 250px (데스크톱), 280px (모바일)
- 드래그 오버: 배경 연하게 강조

### 3. 카드 상세 모달

```
┌─────────────────────────────────┐
│ X                 [공유] [삭제]  │
├─────────────────────────────────┤
│ 제목 (수정 가능)                 │
│                                 │
│ 설명 (markdown 지원)            │
│ 기한: [날짜 선택]               │
│ 우선도: [High/Medium/Low]       │
│                                 │
│ 첨부 파일:                      │
│ [+파일 업로드]                  │
│ - file1.pdf                     │
│                                 │
│ 댓글:                           │
│ ┌─────────────────────────────┐ │
│ │ 박준호: 좋아요!             │ │
│ │ 2시간 전                    │ │
│ └─────────────────────────────┘ │
│ [댓글 입력 창]                  │
│                                 │
│ 활동 로그:                      │
│ - 생성됨 (3일 전)               │
│ - To Do → In Progress (2일 전)  │
│ - 우선도 medium → high (1일 전) │
└─────────────────────────────────┘
```

- 너비: 500px (데스크톱), 100% (모바일)
- 스크롤 가능
- 이름 입력: 간단한 텍스트 (인증 제외)

### 4. 대시보드 위젯

```
┌─────────────────────┐
│ 이번 주 완료    [8] │
├─────────────────────┤
│ 진행 중         [5] │
├─────────────────────┤
│ 가장 활발       👨‍💻 │
│ 박준호          +3   │
├─────────────────────┤
│ 기한 임박       🔴2  │
└─────────────────────┘
```

- 각 통계: 레이블 + 값 (크고 굵음)
- 아이콘/이모지로 한눈에 구분
- 기한 임박: 빨강 경고
- 우측 패널 (데스크톱) / 탭 (모바일)

### 5. 기한 알림

- **12시간 전**: 카드 빨강 / 우측 배너 "⏰ 기한 임박: N개"
- 웹 알림 (선택 가능)
- 배너: 클릭하면 해당 카드 목록 표시

## 인터랙션

### 카드 드래그 & 드롭
- 카드를 다른 컬럼으로 드래그: 실시간 반영 (Supabase)
- 드롭 시 애니메이션: 부드럽게 자리잡기
- 모바일: 터치 드래그 지원

### 카드 완료 애니메이션
- Done으로 이동할 때:
  1. 카드 초록색으로 변함
  2. 불꽃/파티클 1초 애니메이션
  3. "완료!" 텍스트 팝업
  4. 카드가 Done 컬럼에 안착

### 실시간 동기화
- 다른 사용자의 변경사항 즉시 반영 (Supabase Realtime)
- 변경된 카드: 짧은 하이라이트 애니메이션

---

## 접근성

- **색상 외 시각 신호**: 우선도 아이콘 (🔴🟡🟢) + 텍스트
- **키보드 네비게이션**: Tab으로 이동, Enter로 선택
- **명도 대비**: WCAG AA 기준 준수
- **텍스트 크기**: 최소 12px (선택 시 확대 지원)
